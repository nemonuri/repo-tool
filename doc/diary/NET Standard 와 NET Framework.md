# NET Standard 와 NET Framework

- NET Standard 2.0 은, NET Framework 4.6.1 부터 지원한다!
  - NET Standard 2.0 라이브러리가 NET Framework 4.6 을 대상 프레임워크로 빌드했으니, 당연히 문제가 생겼지...
  - .NET 공식 문서에서는, NET Framework 4.7.2 이상을 사용할 것을 권장한다.
